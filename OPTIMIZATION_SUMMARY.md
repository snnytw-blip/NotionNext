## 🎯 利用ガイド

### クイックスタート
```bash
# 開発環境の初期化
npm run init-dev

# 開発サーバーの起動
npm run dev

# 品質チェックの実行
npm run quality
```

### 開発フロー
1. Git フックをインストール: `npm run setup-hooks`
2. 機能を開発
3. コードをコミット (品質チェックが自動実行されます)
4. コードをプッシュ (ビルドテストが自動実行されます)

### デプロイフロー
1. 環境変数を設定
2. `npm run build` でビルドテストを実行
3. デプロイプラットフォームを選択 (Vercel/Netlify/Docker)
4. DEPLOYMENT.md のガイドに従ってデプロイ
